# Rawlabs SEO Skill

## SEO Goal
The website should be built to support organic traffic for freeze-dried cat and dog food searches.

Important themes:
- Freeze dried kedi maması
- Freeze dried köpek maması
- Doğal kedi ödülü
- Doğal köpek ödülü
- Katkısız kedi köpek maması
- Freeze dried mama

## Page-Level SEO Rules
Every important page should have:
- Unique title
- Meta description
- One clear H1
- Logical H2/H3 structure
- Descriptive internal links

Avoid duplicate page titles.

## Product SEO Rules
Each product detail page should support:
- Product name
- Short description
- Detailed description
- Ingredients
- Nutrition values if available
- Usage recommendation
- Storage conditions
- Image alt text

Product detail pages should be readable for both users and search engines.

## Image SEO Rules
All important product images should have meaningful alt text.

Good alt examples:
- Rawlabs freeze dried hamsi ödülü 40g kedi ve köpekler için
- Rawlabs freeze dried tavuk balık köpek maması 300g

Bad alt examples:
- image
- product
- photo
- IMG_1234

## Category SEO Rules
Category pages should clearly indicate what they show.

Examples:
- Kedi Mamaları
- Köpek Mamaları
- Tam Mamalar
- Ödül Mamaları

If URL parameters are used, visible page headings and descriptions should update accordingly.

## Content Rules
Write naturally.
Do not keyword-stuff.
Prioritize clarity and trust.
Use Turkish language for the Turkish website.

## Technical SEO Rules
Keep HTML semantic.
Use descriptive link text.
Avoid empty headings.
Avoid broken links.
Avoid duplicate H1 tags.

## Claims and Trust
Do not use unverifiable claims such as:
- "Veteriner onaylı"
- "%98 memnuniyet"
- "15.000+ mutlu dost"

unless the user confirms that these are real and can be supported.
